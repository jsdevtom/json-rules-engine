stop

  • stop(): this
    • Defined in engine.ts:199

    Stops the rules engine from running the next priority set of Rules. All remaining rules will be resolved as undefined, and no further events emitted. Since rules of the same priority are evaluated in parallel(not series), other rules of the same priority may still emit events, even though the engine is in a "finished" state.

    Returns this
